Features of HTML5

Welcome to my webpage showcasing features of HTML5

Feature 1 : Semantics with New Structural Elements

HTML5 introduces several new semantic elements that provide a clearer structure to web documents. These include:

  • <header> Represents the header of a section or a page.
  • <nav>Defines a set of navigation links.
  • <article>Represents a self-contained piece of content that could be distributed and reused independently.
  • <section>Defines a section in a document.
  • <footer>Represents the footer of a section or a page.

Feature 2 : Audio and Video Embedding

HTML5 makes it easier to embed audio and video directly into web pages using the <audio> and <video> elements.

Feature 3 : Form Validation

HTML5 introduced built-in form validation features, allowing you to validate user input without relying solely on JavaScript. The validation is performed using attributes and pseudo-classes.
